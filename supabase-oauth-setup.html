Backend com Better Auth

Nesta versão, o Supabase fornece somente o PostgreSQL. Cadastro, sessões, confirmação de e-mail, recuperação de senha e OAuth são administrados pelo Better Auth.

Serviços necessários

  • Supabase: banco PostgreSQL
  • Resend: envio de e-mails
  • Google Cloud: login com Google
  • Meta for Developers: login com Facebook
  • LinkedIn Developer Portal: login com LinkedIn
  • Hospedagem Node.js: Railway, Render ou serviço equivalente

Ordem da configuração

  1. Copie .env.example para .env.local.
  2. Configure a conexão PostgreSQL do Supabase.
  3. Execute npm run auth:migrate.
  4. Execute supabase-schema.sql no SQL Editor.
  5. Configure Resend, Google, Facebook e LinkedIn.
  6. Publique o servidor Node.js e atualize as URLs OAuth.
