Configurar login Google e Facebook no Supabase

Use este guia para habilitar o login social no projeto. O site já está preparado para usar os provedores, mas eles precisam estar ativos no Supabase.

1. Criar o projeto

  1. Crie um projeto no Supabase.
  2. Copie a URL do projeto e a chave anon.
  3. Abra o site do Curriculo Pro e cole essas informações no painel Conta.

2. Habilitar o Google

  1. No Supabase, vá em Authentication e depois Providers.
  2. Ative Google.
  3. Adicione o Client ID e o Client Secret do Google Cloud.
  4. Cadastre a URL de retorno autorizada do seu site.

3. Habilitar o Facebook

  1. Em Providers, ative Facebook.
  2. Adicione o App ID e o App Secret do Meta for Developers.
  3. Cadastre a mesma URL de retorno usada no site.

4. Banco de dados

Abra o arquivo supabase-schema.sql e execute no SQL Editor do Supabase para criar a tabela de curriculos.

5. URLs comuns

  • Site local: http://localhost:xxxx
  • Site publicado: use a URL final do deploy, por exemplo GitHub Pages, Netlify ou Vercel.

6. Fluxo esperado

  • A pessoa monta o curriculo.
  • Ao pedir PDF, o modal de acesso aparece.
  • Depois de entrar com Google, Facebook ou e-mail, o PDF é liberado.
