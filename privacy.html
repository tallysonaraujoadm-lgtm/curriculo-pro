Privacidade

O Currículo Pro funciona sem cadastro e sem login. Os dados preenchidos não são enviados para um servidor da aplicação.

O que acontece com os dados

  • Os currículos salvos ficam no armazenamento local do navegador usado.
  • As informações não são sincronizadas entre navegadores ou dispositivos.
  • Limpar os dados do navegador pode excluir os currículos salvos.
  • O PDF é gerado pelo próprio navegador.

Exclusão e segurança

Você pode excluir um currículo pela área “Meus currículos” ou apagar os dados locais do site nas configurações do navegador.

Uso responsável

Evite usar computadores públicos para armazenar informações pessoais. Baixe o PDF e mantenha uma cópia segura.
