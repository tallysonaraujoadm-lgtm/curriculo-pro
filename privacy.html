Privacidade

O Currículo Pro salva os dados no seu navegador para facilitar o uso local. Sem login e sem envio para o servidor, suas informações ficam no dispositivo usado para criar o currículo.

O que acontece com os dados

  • Os campos do currículo podem ser armazenados localmente para serem retomados depois.
  • A foto selecionada também pode ser guardada localmente enquanto o navegador permitir.
  • Ao limpar os dados do navegador, as informações salvas localmente podem ser apagadas.

Uso responsável

Este site foi feito para ajudar na criação de currículos de forma rápida e simples. Se futuramente houver uma conta online ou sincronização, esta política deverá ser atualizada.
