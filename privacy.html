Privacidade

O Curriculo Pro salva os dados no seu navegador para facilitar o uso local. Sem login e sem envio para servidor, suas informacoes ficam no dispositivo usado para criar o curriculo.

O que acontece com os dados

  • Os campos do curriculo podem ser armazenados localmente para retomar depois.
  • A foto selecionada tambem pode ser guardada localmente enquanto o navegador permitir.
  • Ao limpar os dados do navegador, as informacoes salvas localmente podem ser apagadas.

Uso responsavel

Este site foi feito para ajudar na criacao de curriculos de forma rapida e simples. Se futuramente houver conta online ou sincronizacao, esta politica deve ser atualizada.
