Privacidade

O Currículo Pro permite criar uma conta para salvar currículos na nuvem. Os dados ficam associados ao identificador da conta autenticada.

O que acontece com os dados

  • Nome, e-mail, dados de autenticação, sessões e currículos são armazenados para prestar o serviço.
  • Senhas são armazenadas somente em formato criptograficamente protegido pelo Better Auth.
  • Os currículos são mantidos no PostgreSQL e filtrados pelo usuário autenticado.
  • Facebook e LinkedIn podem fornecer dados básicos de perfil quando você escolhe login social.
  • Resend processa os e-mails de confirmação e recuperação de senha.

Exclusão e segurança

Você pode excluir seus currículos pela área “Meus currículos”. Para solicitar exclusão da conta e dos demais dados, use o canal de contato informado pelo responsável pelo site.

Uso responsável

Este site foi feito para ajudar na criação de currículos de forma rápida e simples. Se futuramente houver uma conta online ou sincronização, esta política deverá ser atualizada.
